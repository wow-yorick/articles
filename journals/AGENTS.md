# JOURNALS - AGENTS.md

**Parent:** `../AGENTS.md`

## OVERVIEW

Daily journal entries spanning 2021-2024. Personal reflections, learning progress, and life events in chronological order.

## STRUCTURE

```
journals/
├── 2021_MM_DD.{org,md}    # 2021 entries
├── 2022_MM_DD.{org,md}    # 2022 entries
├── 2023_MM_DD.{org,md}    # 2023 entries
└── 2024_MM_DD.{org,md}    # 2024 entries (sparse)
```

## CONVENTIONS

### File Naming
- Format: `YYYY_MM_DD.{org,md}`
- Months/Days: Not zero-padded (e.g., `2021_7_21.md`)
- Extension: Mixed `.org` and `.md`

### Content Style
- **Outliner format**: Hierarchical bullet points
- **Daily topics**: Learning notes, reflections, events
- **Common themes**:  
  - 有声书 (audiobook narration practice)
  - Programming study
  - Career thoughts
  - Personal finance
  - Life observations

### Task Integration
- Uses Logseq queries to surface tasks
- Markers: `NOW`, `LATER`, `TODO`, `DOING`
- Query pulls tasks from last 14 days

## WHERE TO LOOK

| For | Look In | Pattern |
|-----|---------|---------|
| Specific date | Filename | `2021_07_21.md` |
| Learning topics | Content | Indent under bullet points |
| Tasks | Search | `NOW` or `TODO` markers |
| Life events | Chronological | Browse by date |
